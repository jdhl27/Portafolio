Árbol Binario

Puedes verlo en: Sitio de escritorio

Código en: Árbol-Binario/Codigo

​ La aplicación muestra los nodos graficamente para que la comprención del concepto de árbol binaria sea un poco mas sencilla. Es una estructura de datos en la cual cada nodo puede tener un hijo izquierdo y un hijo derecho. Si algún hijo tiene como referencia a null, es decir que no almacena ningún dato, entonces este es llamado un nodo externo. En el caso contrario el hijo es llamado un nodo interno.
